# you_vimrc
## 目的
使いやすい`vim`の環境を整えること

## 使用にあたって
- 起動前にやること
  - clipboardの設定
    最初に`vim`自体のcripboardの設定を追加する必要がある、環境やバージョンによって違うため割愛
      clipboard設定変更コマンド
      ```
      sudo port deactivate vim
      sudo port install vim +clipboard
      ```
  - w3mのインストール
    `brew install w3m`
